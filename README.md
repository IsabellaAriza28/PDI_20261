# Procesamiento Digital de Imágenes (PDI) 2026-1

Repositorio del curso **Procesamiento de Imágenes Médicas**, donde exploramos técnicas computacionales para analizar imágenes médicas como **CT, MRI, US y rayos X**.

El curso cubre desde **visualización de datos médicos** hasta **segmentación y análisis automatizado**.

---

# Objetivos del curso

Al finalizar este curso podrás:

- Entender cómo funcionan las imágenes médicas
- Leer y manipular archivos **DICOM** y **NIfTI**
- Visualizar volúmenes médicos 2D y 3D
- Aplicar técnicas de procesamiento de imagen
- Implementar algoritmos de visión por computadora
- Analizar datasets médicos reales

---

# Tecnologías utilizadas

El curso utiliza principalmente **Python** y el ecosistema científico.

Principales librerías:

- numpy
- matplotlib
- scikit-image
- pydicom
- SimpleITK
- nibabel
- scipy

---

# Software recomendado

Para visualizar y explorar imágenes médicas se recomienda instalar alguno de los siguientes visores:

### Visores de imágenes médicas

- **3D Slicer**  
  https://www.slicer.org  
  Plataforma open-source muy utilizada para visualización, segmentación y análisis 3D.

- **MICRODICOM**  
  https://www.microdicom.com/  
  Visor DICOM rápido y ligero multiplataforma.

- **RadiAnt DICOM Viewer**  
  https://www.radiantviewer.com  
  Visor DICOM rápido y ligero (Windows).
